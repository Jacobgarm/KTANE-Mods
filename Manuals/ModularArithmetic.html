On the Subject of Modular Arithmetic

Get it?

  • To solve the module, a number, x, must be determined using a series of calculations invloving modular arithmetic. x will be obtained using the chinese remainder theorem on 3 congruences.
  • The moduli of the congruences are given by which LED's are flashing. From right to left, the LED's correspond to the first five primes.
    • The red LED is the moduli of the first congruence.
    • The green LED is the second.
    • The blue LED is the last.
    The moduli are denoted p1, p2 and p3.
  • The number of times the LED's flash per cycle is called F. The symbol DS denotes the digit sum of the digits in the serial number.
  • To input the number x, convert it to ternary and press the three buttons in the correct sequence.

The congruences

First congruence x ≡ FDS (mod p1)
Second congruence x ≡ F-1 (mod p2)
Third congruence x ≡ F + DS + (F+DS)? (mod p3)
where x? denotes the Legendre symbol with respect to p3.

Appendix A: Number theory

The chinese remainder theorem

Legendre symbol

The Legendre symbol, denoted here as x? and more commonly as a parenthesized fraction is a number that describes whether a given number is a quadratic residue modulo another. Given an odd prime p, the number x is a quadratic residue if the congruence
    a2 ≡ x (mod p)
Has a solution. If this is the case, the value of the Legendre symbol is 1. If this is not the case, the value is -1. In the special case where p | x, the value of the symbol is zero. When calculating Legendre symbols, one can employ brute-force for small inputs. However for larger values, the following theorem due to Euler is practical.
    x? ≡ x(p - 1) / 2 (mod p)
